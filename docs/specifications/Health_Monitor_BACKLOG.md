# Health Monitor — Backlog

Этот файл содержит идеи, которые не входят в MVP, но могут быть добавлены позже.

## Справочник симптомов

### Менеджмент справочника симптомов

В MVP список симптомов может быть статическим.

В будущем можно добавить полноценный менеджмент справочника:

- страница `/settings/symptoms`;
- добавление нового симптома;
- переименование симптома;
- скрытие неактуального симптома;
- сортировка симптомов;
- группировка симптомов по категориям;
- цвет или иконка для симптома;
- защита от дублей.

Пример категорий:

- ЖКТ;
- сон;
- стресс;
- активность;
- питание;
- общее состояние.
