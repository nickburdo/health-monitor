## План разработки

### Этап 1. Инициализация проекта

- создать Nuxt 4 проект;
- подключить TypeScript;
- подключить Nuxt UI;
- настроить Tailwind;
- настроить ESLint / Prettier.

### Этап 2. База данных

- установить Prisma;
- настроить SQLite;
- создать `schema.prisma`;
- выполнить миграцию;
- проверить базу через Prisma Studio.

### Этап 3. API

- реализовать API для глюкозы;
- реализовать API для давления;
- реализовать API для веса;
- реализовать API для симптомов;
- реализовать ignore / restore.

### Этап 4. UI каркас

- сделать layout;
- сделать header/sidebar;
- добавить кнопку быстрого ввода;
- сделать модальное окно.

### Этап 5. Формы

- форма глюкозы;
- форма давления;
- форма веса;
- форма симптома.

### Этап 6. Таблицы

- таблица глюкозы;
- таблица давления;
- таблица веса;
- таблица симптомов;
- кнопки Ignore / Restore.

### Этап 7. Графики

- график глюкозы;
- график давления;
- график веса;
- график симптомов.

### Этап 8. Dashboard

- последние значения;
- карточки;
- графики;
- фильтры периода.

### Этап 9. Полировка

- пустые состояния;
- loading states;
- обработка ошибок;
- уведомления;
- адаптивность.
